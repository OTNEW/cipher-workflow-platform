=======================
cipher
======================

Easy-to-use, parallel, and through data processing workflow platform for next-generation sequencing datasets
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

cipher addresses the challenge of building, maintaining, and installing various pipelines for different genomic datasets
